## Overview
This project combines Python‑based Exploratory Data Analysis (EDA) and an interactive Power BI dashboard to analyze retail store performance.
It uncovers insights into customer demographics, seasonal trends, and product sales to help optimize retail strategies.

## Objectives
Perform EDA on retail sales data using Python libraries.
Visualize insights through Power BI dashboard filters (Category, Gender, Age, Season).
Identify key metrics such as total sales, average sale, and top locations.

## Tools & Technologies
Python (Pandas, NumPy, Matplotlib, Seaborn)	Data cleaning, transformation, and visualization
Power BI	Dashboard creation and interactive analytics
Jupyter Notebook	EDA documentation and code execution
CSV Dataset (retail_sales.csv)	Source data for analysis

## EDA Process
Import Libraries – Pandas, NumPy, Matplotlib, Seaborn.
Load Dataset – Read retail_sales.csv and preview data.
Data Cleaning – Handle missing values, correct data types, remove duplicates.
Exploratory Analysis – Analyze sales by category, gender, size, and season.
Visualization – Generate plots for revenue distribution and seasonal trends
